Course Registration

JavaFX desktop app for browsing a course catalog and managing course reviews. Supports searching by subject/number/title, viewing average ratings, and CRUD actions on reviews through an FXML-based UI backed by SQLite.

GitHub Repo

Highlights

Java JavaFX FXML SQLite MVC DAO
  • Course search and browsing UI built with JavaFX + FXML.
  • Review workflow: submit, edit, delete, and compute average course rating.
  • Persistence layer using SQLite with a database driver and DAO pattern.
